Description:
This project was carried out in the course of cross-platform application development. The project consisted of an ERP that allowed to manage the technical service of a store of computer science.

Functionality:

  • Customer Management
  • Incident management

Technologies:
  • HTML
  • Bootstrap
  • Spring Boot
  • Spring Data
  • Spring Security
  • Java
  • Lombok
  • Thymeleaf
  • MySQL
  • Git

ERP Objectives:
The ERP has the objective of managing the technical service department of a store of computer science. We try to give workers speed, offering a simple and intuitive system to manage your tasks.

Needs to be solved:
With the ERP we will improve the management and operation of the technical service department, facilitating a more organized and effective work to the workers of the company.
