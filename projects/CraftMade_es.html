Descripción:
Este proyecto se realizó en el curso de desarrollo de aplicaciones web. El proyecto consistía en un MarketPlace por artesanos que quieran vender sus productos de manera en línea.

Tecnologías:

  • HTML
  • CSS
  • SASS
  • JavaScript
  • MariaDB
  • Php
  • Laravel
  • Vagrant
  • Git

Objetivos de la aplicación:
El objetivo de esta práctica es poder ofrecer tu producto, de manera en línea. Es decir, un artesano, tendrá uno varios productos, que ofrecerá en nuestra página web. El cliente podrá navegar por toda la web en busca del producto deseado. Nuestra aplicación web será capaz de filtrar los productos en diferentes categorías, para poder facilitar la busca del usuario. Esta idea ha surgido, en busca de todo pequeño artesano que no tiene posibilidad de poder ofrecer su producto de manera en línea.
