Descripción:
Este proyecto se realizó en el curso de desarrollo de aplicaciones multiplataforma. El proyecto consistía en un ERP que permitía gestionar el servicio técnico de una tienda de informática.

Funcionalidades:

  • Gestión de clientes
  • Gestión de incidencias

Tecnologías:
  • HTML
  • Bootstrap
  • Spring Boot
  • Spring Data
  • Spring Security
  • Java
  • Lombok
  • Thymeleaf
  • MySQL
  • Git

Objetivos del ERP
El ERP tiene el objetivo de gestionar el departamento de servicio técnico de una tienda de informática. Tratamos de brindar a los trabajadores velocidad, ofreciendo un sistema simple e intuitivo para administrar sus tareas.

Necesidades a resolver:
Con el ERP mejoraremos la gestión y funcionamiento del departamento de servicio técnico, facilitando un trabajo más organizado y eficaz a los trabajadores de la empresa.

A quien está dirigido:
El ERP está dirigido a los trabajadores de empresas la función de las cuales sea hacer el mantenimiento o reparar: portátiles, ordenadores o cualquier periférico; los cuales presenten averías o problemas casuales que harán que el funcionamiento de la maquinaria no sea el adecuado.
