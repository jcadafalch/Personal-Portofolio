HardwaRepair - ERP

  • Projecte GitHub: Enllaç

Descripció:
Aquest projecte es va realitzar en el curs de desenvolupament d'aplicacions multiplataforma. El projecte consistia en un ERP que permetia gestionar el servei tècnic d'una botiga d'informàtica.

Funcionalitats:

  • Gestió de clients
  • Gestió de incidències

Tecnologies:
  • HTML
  • Bootstrap
  • Spring Boot
  • Spring Data
  • Spring Security
  • Java
  • Lombok
  • Thymeleaf
  • MySQL
  • Git

Objectius de l'ERP:
El ERP té l'objectiu de gestionar el departament de servei tècnic d'una botiga d'informàtica. Tractem de brindar als treballadors velocitat, oferint un sistema simple i intuïtiu per a administrar les seves tasques.

Necessitats a resoldre:
L'ERP està dirigit als treballadors d'empreses la funció de les quals sigui fer el manteniment o reparar: portàtils, ordinadors o qualsevol perifèric; els quals presentin avaries o problemes casuals que faran que el funcionament de la maquinària no sigui l'adequat.
