GExpenses - Aplicación web

  • Proyecto GitLab: Enlace

Descripción:
Este proyecto se realizó en el curso de desarrollo de aplicaciones web. El proyecto consistía en una aplicación web por creación de actividad, de forma que puedes llevar un registro de los gastos, quién ha pagado, hay quién se tiene que pagar, en resumen te facilita las cuentas de cualquier actividad.

Tecnologías:

  • HTML
  • CSS
  • JavaScript
  • MariaDB
  • Php
  • Vagrant
  • Git

Objetivos de la aplicación:
El objetivo de esta práctica es, el desarrollo de un gestor de gastos denominado GExpenses. La finalidad es poder repartir los gastos con los integrantes. Es decir todos los gastos que puedan surgir en un viaje, por ejemplo, si feudo una compra para hacer la comida, y esta compra la paga un integrante del grupo, que se pueda repartir los gastos por todos los integrantes por partes iguales. Con esta explicación web, buscamos la facilidad de poder hacer un viaje, con un conjunto de personas, y que la complicación en la hora de hacer el reparto de gastos la facilite esta aplicación.
